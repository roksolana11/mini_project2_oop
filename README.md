# Kaggle Courses — Roksolana Blashchuk

Цей репозиторій містить код та вправи з трьох курсів, пройдених на платформі [Kaggle Learn](https://www.kaggle.com/learn).

## Курси

### 1. [Intro to Machine Learning](https://www.kaggle.com/learn/intro-to-machine-learning)
Вступний курс з машинного навчання. Охоплює побудову перших моделей за допомогою scikit-learn, роботу з даними через pandas, оцінку якості моделі (MAE), концепції overfitting/underfitting та алгоритм Random Forest.

### 2. [Intermediate Machine Learning](https://www.kaggle.com/learn/intermediate-machine-learning)
Курс про роботу з реальними даними. Охоплює обробку пропущених значень, кодування категоріальних змінних, побудову пайплайнів, крос-валідацію, алгоритм XGBoost та виявлення витоку даних (Data Leakage).

### 3. [Data Visualization](https://www.kaggle.com/learn/data-visualization)
Курс з візуалізації даних за допомогою бібліотеки seaborn. Охоплює лінійні графіки, стовпчасті діаграми, теплові карти, діаграми розсіювання та гістограми.

## Структура репозиторію
Course 1 (Introduction To Machine Learning)\
Course 2 (Intermediate Machine Learning)\
Course 3 (Data Visualization)
